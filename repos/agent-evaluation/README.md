# Agent Evaluation

Agent 性能测试和评估框架。

## 定义

Agent Evaluation 框架提供：
- **基准测试**：Agent 评估的标准化任务和指标
- **测试**：Agent 行为的单元和集成测试
- **监控**：运行时可观测性和性能跟踪
- **比较**：跨框架和跨模型比较
